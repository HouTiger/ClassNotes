### 3  
先求有n个人，每个人有两种估值$a, b，(a > b)$，且选择两种估值的概率均为$1/2$  
求卖家的收益期望 

| 组合      | 收益 | 概率                    |
|-----------|------|-------------------------|
| 全为b     | b    | $1/2^n$                 |
| 只有1个a  | b    | $n/2^n$                 |
| 多于两个a | a    | $1 - \frac{n + 1}{2^n}$ |  
总收益为$R(a, b, n) = a - \cfrac{(a-b)(n + 1)}{2^n}$   
* (1)
a = 1, b = 0, n = 2  
代入上式，收入期望为1/4   
* (2)  
a = 1, b = 0, n = 3  
收入期望为 1/2  
* (3)  
$R(1, 0, n) = 1 - \cfrac{n + 1}{2^n}$  
当n逐渐增加，收益期望逐渐逼近1
